# Emaily
A full-stack web application that allows users to create and send survey emails, track responses, and manage email campaigns through a credit-based system.

# Features
- Create and send email surveys
- Track responses (Yes/No results)
- Credit-based email sending system
- Payment integration for purchasing credits
- Dashboard to view sent surveys and results
- Clean and responsive UI

# Tech Stack
- Frontend: React, Materialize CSS
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: Google OAuth
- Payments: Stripe API

# Installation
1. Clone the repository

```bash
git clone https://github.com/your-username/emaily.git
```

2. Navigate into the project

```bash
cd emaily
```

3. Install dependencies

```bash
npm install
cd client
npm install
```

4. Run the app

```bash
npm run dev
```

---


# Environment Variables
Create a `config` or `.env` file and add:

```env
GOOGLE_CLIENT_ID=your_key
GOOGLE_CLIENT_SECRET=your_secret
MONGO_URI=your_mongo_connection
STRIPE_KEY=your_stripe_key
```
